**Служба такси**

**Почему востребовано:** Альтернатива дорогим агрегаторам, возможность запустить собственный сервис такси.[20][21][22]

**Ключевой функционал для демо:**
- Карта с отображением текущего местоположения
- Ввод адресов «Откуда» и «Куда» с автоподсказками
- Выбор класса автомобиля (эконом, комфорт, бизнес)
- Расчет стоимости и времени поездки
- Дополнительные опции (детское кресло, багаж)
- Отслеживание водителя в реальном времени
