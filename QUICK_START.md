# Быстрый старт

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Использование

1. Введите адрес отправления в поле "Откуда?"
2. Введите адрес назначения в поле "Куда?"
3. Выберите класс автомобиля
4. При необходимости выберите дополнительные опции
5. Нажмите "Заказать такси"
6. Следите за статусом водителя

## Тестовые данные

При вводе адресов используйте:
- "Москва" для автоподсказок
- Или выберите из сохраненных: "Дом" или "Работа"
